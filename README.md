## **VSCode**

Required plugins:

```
1 - ESlint
2 - Prettier - Code formatter
3 - EditorConfig for VS Code
```

<br />

## Getting Started

<br />

`1 -` Duplicate the `.env-example` file and rename it to `.env` and complete the information.

`2 -` Run the following command on your local environment:

```bash
npm install
npm run dev
```
